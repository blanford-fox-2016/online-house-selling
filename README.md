# online-house-selling

## Model's Structure
```json
{
  "title"       : String,
  "description" : String,
  "photo"       : String,
  "price"       : Number,
  "location"    : {
    "address"         : String,
    "addressCountry"  : String,
    "postalCode"      : String,
    "long"            : String,
    "lat"             : String,
  },
  "createdAt"   : Date,
  "updatedAt"   : Date
}
```
